Homework Git

Lesson new team

  • git status - перевірка статусу
  • git branch - перевірка найменування гілок
  • git remote-v - розташування на данний момент
  • git add . - збереження в поточну папку
  • git config --global user.name - ідентифікатор імені
  • git config --global user.email - ідентифікатор email
  • git commit -m - опис що було зроблено
  • git log - перевірка хто що зробив
  • git push origin HEAD - синхронізація з main
  • git checkout -b - створення паралельної гілки
